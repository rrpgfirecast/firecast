#Ficha da mesa Konoha Heroes

O sistema é próprio e levemente arcaico (erros de digitação, português e até alguns jutsus e itens mais fortes que outro), se alguem algum dia tiver interesse, talvez seja possivel repassar o sistema. 

Modelo original: Log Horizon aqui neste git mesmo.
Adaptação/modificação para o modelo Konoha Heroes por HashBR (Túlio Campos).
e-mail para contato: tulio.bart at msn dot com

Maioria dos campos relevantes são calculados sozinhos.

Arte dos créditos feita por Átilla.

##Update do dia 02/10/2017

-Duas abas foram removidas e agora há uma única aba para jutsus, mais compacta, muito mais bonita visualmente e com campo dano.

-Campo Efeitos alterado para Bônus dos Atributos.

-Anotações agora é um node para que possa ser modificado a permissão pelo mestre, assim dando direito de escrever nestes campos ao dono da ficha.

-Bandana de ninja da vila vs ninja renegado agora é só um imagecheckbox, não havia necessidade de puxar imagens da internet como se fosse várias imagens.

-Página principal é melhor de se ver quando não se tem os direitos de dono/mestre.
