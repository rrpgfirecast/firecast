# Animon Story - Ficha para Firecast

Plugin de ficha para o sistema **Animon Story** no Firecast.

## Estado atual

Esta é uma versão inicial da ficha, criada com foco em reproduzir a estrutura e organização da ficha oficial em PDF dentro do Firecast.


## Funcionalidades atuais

- Campos editáveis da ficha
- Estrutura baseada na ficha oficial

## Planejamento futuro

Funcionalidades planejadas:

- Rolagens automáticas pelo chat
- Cálculos automáticos
- Botões de testes e ações
- Recursos e atributos dinâmicos
- Melhorias visuais
- Reformulação completa da interface
- Melhor adaptação para uso durante sessões
