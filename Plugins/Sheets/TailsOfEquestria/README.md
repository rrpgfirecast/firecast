# Projeto Firecast

Esta pasta concentra os arquivos executáveis e gerados da ficha storybook:

- `module.xml`: manifesto do módulo;
- `frmTailsOfEquestriaMain.lfm`: livro de aventura principal;
- `frmTailsTalentItem.lfm`: cards de talentos;
- `frmTailsQuirkItem.lfm`: cards de caprichos;
- `frmTailsInventoryItem.lfm`: cards do Alforge;
- `frmTailsFriendItem.lfm`: cards do Mapa de Amizades;
- `frmTailsTimelineItem.lfm`: cards dos episódios;
- `sdk/`: stubs locais gerados por `rdk p`;
- `output/`: saída gerada pelo compilador.

Execute `rdk i` a partir desta pasta para compilar e instalar.
